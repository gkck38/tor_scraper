## File Descriptions

### Core Code Files

#### main.go (430 lines)
- **Language:** Go 1.21+
- **Purpose:** Complete Tor scraper application
- **Modules:**
  1. File reading with error handling
  2. Tor SOCKS5 proxy configuration
  3. HTTP request with timeout management
  4. Report generation (JSON + log + HTML)
- **Key Functions:** readTargets, createTorClient, scanURL, saveScanReport, main
- **Dependencies:** golang.org/x/net/proxy

#### go.mod (5 lines)
- **Purpose:** Go module definition
- **Contains:** Module path and Go version requirement
- **Auto-Updated:** When running `go get`

#### go.sum (auto-generated)
- **Purpose:** Dependency lock file
- **Contains:** SHA256 hashes of dependencies
- **Auto-Generated:** By Go tool during `go mod tidy`

### Configuration Files

#### targets.yaml (Sample)
- **Format:** YAML/Text
- **Purpose:** List of .onion addresses to scan
- **How to Edit:** Add one URL per line
- **Example Contents:**
  ```yaml

  3g2upl4pq6kufc4m.onion
  thehiddenwiki.onion
  example.onion
  ```

## Compilation & Execution

### Building
```bash
# Option 1: Direct execution
go run main.go targets.yaml

# Option 2: Build executable
go build -o tor-scraper.exe main.go
./tor-scraper.exe targets.yaml

# Option 3: Cross-platform build
GOOS=windows GOARCH=amd64 go build -o tor-scraper.exe main.go
GOOS=linux GOARCH=amd64 go build -o tor-scraper main.go
GOOS=darwin GOARCH=amd64 go build -o tor-scraper main.go
```

### Execution
```bash
# Windows
tor-scraper.exe targets.yaml

# Linux/Mac
./tor-scraper targets.yaml

# With custom output directory
go run main.go targets.yaml /path/to/output
```
